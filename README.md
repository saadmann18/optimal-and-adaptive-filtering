# Optimal and Adaptive Filtering - Course Tutorials

**University:** Brandenburg University of Technology Cottbus-Senftenberg  
**Semester:** Winter Semester 2020/2021  
**Period:** October 2020 - March 2021  

## Course Overview

This repository contains practical implementations and tutorials for the Optimal and Adaptive Filtering course. The course covers fundamental and advanced techniques in signal processing, focusing on filter design, optimization, and adaptive algorithms.

## Tutorial Structure

The course is organized into 6 main tutorials, each building upon previous concepts:

### T1: Bayesian Parameter Estimation
- **Topics:** Bayesian inference, parameter estimation, posterior distributions
- **Files:** Python templates and solutions for Bayesian estimation algorithms
- **Skills:** Statistical inference, probabilistic modeling

### T2: Wiener Filter
- **Topics:** Optimal filtering, mean-square error minimization, linear prediction
- **Files:** Wiener filter implementation and experimental analysis
- **Skills:** Filter design, signal denoising, prediction

### T3: Deterministic Optimization
- **Topics:** Gradient descent, Newton's method, convergence analysis
- **Files:** Optimization algorithms and convergence studies
- **Skills:** Numerical optimization, algorithm analysis

### T4: Stochastic Gradient
- **Topics:** Stochastic gradient descent, adaptive algorithms, audio processing
- **Files:** SGD implementations with real audio data
- **Skills:** Adaptive filtering, real-world signal processing

### T5: Least Squares
- **Topics:** Linear least squares, recursive least squares, system identification
- **Files:** LS algorithms and numerical examples
- **Skills:** System identification, parameter estimation

### T6: Kalman Filter
- **Topics:** State estimation, recursive Bayesian filtering, tracking
- **Files:** Kalman filter implementation with tracking examples
- **Skills:** State-space modeling, dynamic system estimation

## Learning Progression

The tutorials follow a logical progression:

1. **Foundation (T1-T2):** Statistical estimation and optimal filtering
2. **Optimization (T3-T4):** From deterministic to stochastic optimization
3. **Advanced Methods (T5-T6):** Recursive algorithms and state estimation

## Prerequisites
